# Space Invaders

A close recreation of the original 1978 Space Invaders arcade game, written in JavaScript. Built as a final project for an intro to computer science class in 7th grade, where it was named the most complex and most interesting submission by the instructor.

## Features

- Three enemy types with different point values and animations
- Shields that degrade as they take hits
- Bonus ship that flies across the top periodically
- Three simultaneous enemy bullet types
- Score tracking and lives system
- Enemy swarm speeds up as aliens are eliminated
- Starfield background

## How to run

The original code was written for Code.org's Game Lab environment. To run it, paste the contents of `space_invaders_recreation_js_code.js` into a new project at [studio.code.org/projects/gamelab](https://studio.code.org/projects/gamelab) and hit Run.

Once inside:

- Arrow keys to move
- Space or Up arrow to shoot
